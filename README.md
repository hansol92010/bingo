### :pushpin:Description
1. 5X5 빙고판에는 1부터 25까지 랜덤으로 추출한 수가 각 칸마다 출력된다
2. 기본적으로 색상은 "yellow"이지만 클릭 후에는 "pink"로 변한다
3. 가로 혹은 세로, 대각선의 연속된 5개의 칸이 "pink"로 변하면 빙고가 되어 빙고 수가 1씩 올라간다
4. 가로 빙고, 세로 빙고, 대각선 빙고가 되면 해당 빙고는 비활성화 된다
